# Accounts Payable Data Analysis Using SQL

## Objective
The objective of this project is to analyze Accounts Payable data using SQL to gain insights into

## Dataset Description
Dataset Used: Accounts-Payable.csv
It contains information about
- vendor details
- Invoice amounts
- Due dates
- Payment dates
- Payment status
- Delay Days

## Tools Used
- SQL 
- SQL Server
- CSV (Data Source)

## Analysis Performed
The following analysis was performed using SQL
- Calculation of total overdue payments
- Payment delay calculation using DATEDIFF
- Vendor-wise total spending
- Vendor risk analysis based on payment delays
- Joins between Accounts Payable and Vendor table

## Key Insights
- Identified vendors with highest payment delays
- Found total overdue payable amount in the dataset
- Vendor wise average delay
